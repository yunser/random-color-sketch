# sketch-yunser

* 说明：Sketch 随机颜色插件
* 功能
  * 选中图层后快速设置颜色。
* TODO
  * 支持 React
* 参考

## 直接使用

[下载页面](https://github.com/yunser/random-color-sketch/releases)


## 开发


```
npm i
npm run watch

skpm log -f
```

npm run build，然后对 `sketch-yunser.sketchplugin` 目录进行打包。
